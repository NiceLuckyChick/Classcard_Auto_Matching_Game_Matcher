# Classcard_Auto_Matching_Game_Matcher
클래스카드 매칭 게임 전용 툴


클래스카드(Classcard)의 단어 세트 데이터를 자동으로 수집하고, 이를 기반으로 매칭 게임을 자동으로 플레이해 주는 셀레니움(Selenium) 기반의 파이썬 스크립트입니다.

🛠️ 요구 사항 및 준비물
이 프로그램을 실행하기 위해서는 아래의 환경이 필요합니다.

Python 3.x 버전 이상

Google Chrome 브라우저 (최신 버전 권장)

Selenium 라이브러리

🚀 설치 및 실행 방법
1. 관련 라이브러리 설치
터미널(Terminal) 또는 명령 프롬프트(CMD)를 열고 아래 명령어를 입력하여 셀레니움을 설치합니다.

Bash
pip install selenium
2. 스크립트 실행
파이썬 파일(예: main.py)을 실행하면 터미널에 로그인 정보를 입력하는 창이 뜹니다. 본인의 클래스카드 계정 정보를 입력하세요.

Plaintext
ID : [본인의 클래스카드 아이디 입력]
PWD : [본인의 클래스카드 비밀번호 입력]
3. 데이터 수집 및 매칭 게임 진행 (★ 수동 조작 필요)
프로그램은 로그인까지만 자동으로 진행합니다. 이후에는 브라우저를 보며 직접 조작해야 합니다.

세트 페이지 진입: 로그인이 완료되면 학습하고자 하는 단어 세트의 메인 페이지로 직접 클릭해서 이동합니다. (URL에 /set/이 포함되면 봇이 자동으로 데이터를 수집하여 json 파일로 저장합니다.)

매칭 게임 시작: 데이터 수집 완료 메시지(Data Saved.)가 뜨면, 매칭 게임을 직접 클릭해서 시작합니다. 게임이 시작되면 봇이 자동으로 정답 카드를 매칭하기 시작합니다.

✨ 주요 기능
자동 로그인: 입력한 ID/PWD를 통해 클래스카드 사이트에 자동으로 접속합니다.

데이터 자동 학습 및 저장: 단어 세트 페이지 방문 시 텍스트 단어, 뜻뿐만 아니라 오디오 데이터(data-src)까지 추출하여 cardbot_Source.json 및 classcard_audio.json 파일로 저장합니다.

텍스트 & 오디오 매칭 지원: 일반 텍스트 매칭뿐만 아니라, 귀로 듣고 맞추는 오디오 매칭 게임 모드도 자동으로 인식하여 정답을 클릭합니다.

실시간 상태 표시: 터미널 창을 통해 현재 매칭 횟수(Count)와 점수(Score)를 실시간으로 확인할 수 있습니다.

⚠️ 주의 사항 및 팁
💡 최신 셀레니움 버전 기준: 본 코드는 크롬 드라이버를 자동으로 관리하므로 별도의 chromedriver.exe 다운로드가 필요하지 않습니다. 단, 에러가 발생할 경우 크롬 브라우저 버전과 일치하는 드라이버 파일을 스크립트와 같은 폴더에 넣어주세요.

🎨 터미널 환경 권장: 코드 내에 터미널 색상 출력 기능(RED, GREEN, YELLOW)이 포함되어 있습니다. 윈도우 기본 명령 프롬프트(cmd)에서는 글자가 깨질 수 있으므로, VS Code 내장 터미널이나 Windows Terminal 환경에서 실행하는 것을 강력히 권장합니다.
